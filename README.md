# paygentic-sdk

Python SDK for the [Paygentic API](https://docs.paygentic.io).

## Installation

```bash
pip install paygentic-sdk
```

## Usage

```python
from paygentic_sdk import Paygentic

sdk = Paygentic(api_key="your-api-key")
```

For full documentation, see [docs.paygentic.io](https://docs.paygentic.io).
