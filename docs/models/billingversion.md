# BillingVersion

Billing engine version. 0 = legacy fee-schedule billing (Legacy), 1 = line-item billing with metered usage support (Standard).

## Example Usage

```python
from paygentic_sdk.models import BillingVersion
value: BillingVersion = 0
```


## Values

| Name   | Value  |
| ------ | ------ |
| `ZERO` | 0      |
| `ONE`  | 1      |
