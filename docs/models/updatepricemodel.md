# UpdatePriceModel

The pricing model to be used, which can be standard, dynamic, volume-based, or percentage-based.

## Example Usage

```python
from paygentic_sdk.models import UpdatePriceModel
value: UpdatePriceModel = "standard"
```


## Values

| Name         | Value        |
| ------------ | ------------ |
| `STANDARD`   | standard     |
| `DYNAMIC`    | dynamic      |
| `VOLUME`     | volume       |
| `PERCENTAGE` | percentage   |
