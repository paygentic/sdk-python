# RedirectUrls

Optional redirect URLs after payment completion or failure. If not provided, uses default platform behavior.


## Fields

| Field                                                                                                                                                                          | Type                                                                                                                                                                           | Required                                                                                                                                                                       | Description                                                                                                                                                                    |
| ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `on_success`                                                                                                                                                                   | *Optional[str]*                                                                                                                                                                | :heavy_minus_sign:                                                                                                                                                             | Redirect destination after successful payment. Sample values: 'https://app.example.com/success', 'https://dashboard.company.com/welcome', 'https://portal.startup.io/complete' |
| `on_failure`                                                                                                                                                                   | *Optional[str]*                                                                                                                                                                | :heavy_minus_sign:                                                                                                                                                             | Redirect destination after failed payment. Sample values: 'https://app.example.com/payment-failed', 'https://dashboard.company.com/retry', 'https://portal.startup.io/error'   |
